# 最长公共子串问题

最长公共子串问题是指在两个或多个字符串中找到最长的字符串（或多个字符串）作为它们的子串。

## 示例

字符串 `ABABC`、`BABCA` 和 `ABCBA` 的最长公共子串是长度为 3 的字符串 `ABC`。其他公共子串包括 `A`、`AB`、`B`、`BA`、`BC` 和 `C`。

```
ABABC
  |||
 BABCA
  |||
  ABCBA
```
