# 回文检查（Palindrome Check）

回文是一个正向和反向读取相同的字符串。这意味着字符串的后半部分是前半部分的逆序。

## 示例

以下字符串是回文（将返回 `TRUE`）：

```
- "a"
- "pop"     ->  p + o + p
- "deed"    ->  de + ed
- "kayak"   ->  ka + y + ak
- "racecar" ->  rac + e + car
```

以下字符串不是回文（将返回 `FALSE`）：

```
- "rad"
- "dodo"
- "polo"
```
