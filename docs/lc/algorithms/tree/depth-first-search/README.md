# 深度优先搜索（DFS）

深度优先搜索（DFS）是一种用于遍历或搜索树或图数据结构的算法。它从根节点开始（在图的情况下选择一些任意节点作为根节点），沿着每条分支尽可能远地探索，直到无法继续为止，然后回溯。
