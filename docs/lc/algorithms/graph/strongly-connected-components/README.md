# 强连通分量

如果有向图中任意一对顶点之间存在来回的路径，那么该有向图被称为**强连通**图。在一个可能不是强连通的有向图 G 中，如果顶点 `u` 和 `v` 之间存在来回的路径，那么它们被称为彼此强连通。
