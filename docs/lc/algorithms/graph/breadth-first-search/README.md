# 广度优先搜索（BFS）

广度优先搜索（BFS）是一种用于遍历和搜索树或图数据结构的算法。它从树根（或图的某个任意节点，有时称为“搜索键”）开始，先探索邻居节点，然后再移动到下一级邻居。
